# CRM Simples

Esse projeto nasceu de uma vontade de voltar a codar. Fiquei um tempo parado
e senti falta — programar é algo que eu gosto de verdade, e nada melhor do
que retomar com um projeto que tem utilidade real.

A ideia foi simples: criar um sistema de cadastro de clientes que qualquer
comércio pequeno pudesse usar. Barbearia, salão, oficina — todo mundo tem
clientes e quase ninguém controla direito.

## O que o sistema faz

Cadastra clientes com nome, telefone, e-mail e observações. Você consegue
ver todos na tabela e deletar quando precisar.

## Tecnologias que usei

Desenvolvi com Java 21 e Spring Boot, usando Spring Data JPA para o banco
de dados e H2 para armazenamento. A tela foi feita com HTML, CSS e
JavaScript puro, sem framework.

## Como rodar

Clone o repositório, abra no IntelliJ e rode a classe `CrmSimplesApplication`.
Depois é só acessar `http://localhost:8080` no navegador.

## Dificuldades

A parte mais difícil foi relembrar algumas sintaxes e a lógica de orientação
a objetos — mas nada que a prática não resolva.

## Próximos passos

Ainda pretendo adicionar edição de clientes, migrar para MySQL e colocar
o sistema online para que qualquer pessoa possa acessar.
